Real-Time Weather Data Pipeline

The project implements a real-time weather data pipeline using modern data engineering tools. A Python Kafka producer fetches live weather data from an external API and streams it to Apache Kafka. Apache Spark Streaming processes the data in real time and writes it to a PostgreSQL database and pgAdmin is used to manage and query the PostgreSQL database. The processed data is visualized using a Streamlit dashboard for interactive insights. Docker Compose orchestrates all services including Kafka, Spark, PostgreSQL, and Streamlit.
